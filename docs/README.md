# Documentación

Este directorio contiene guías específicas para el proyecto **Knowledge_Definitive**.

- `ACCESSIBILITY.md`: Pautas WCAG 2.2 aplicadas al frontend, controles de accesibilidad y recomendaciones para nuevas funcionalidades.
- `API.md`: Descripción de endpoints REST, esquemas de request/response y consideraciones de seguridad.
- `QA_CHECKLIST.md`: Lista de verificación para QA manual, incluyendo accesibilidad, seguridad y performance.

> Mantenga estos documentos actualizados conforme evolucione la plataforma.
